vscode安装及使用教程

一、安装vscode

1、到vscode官网https://code.visualstudio.com/docs/?dv=win下载vscode并安装

2、将语言切换为中文简体：打开vscode，按下组合键Ctrl+Shift+P，搜索configure display language，点击进入，然后点击Install additional languages...，找到中文简体，点击install安装中文简体语言，安装完成重启即可

3、新建一个项目，在D盘新建一个vscodeworkspace文件夹，在vscodeworkspace下新建一个名为project01的项目目录，打开vscode，打开文件夹，选择刚刚新建的project01目录，打开完了之后，新建一个文件，名为index.html，然后在index.html中输入!，按下Tab键声明文档类型为html5类型文件

4、更改字体大小，文件-->首选项-->设置，可以字体以及字体大小

5、使用Ctrl+鼠标滚轮更改字体大小，文件-->首选项-->设置-->功能-->扩展，点击：在settings.json中编辑，在settings.json中添加一行："editor.mouseWheelZoom": true,

6、安装atom的one dark主题，文件-->首选项-->颜色主题，点击：安装其他颜色主题，输入one dark，找到one dark pro安装即可

7、在浏览器中打开html文件，点击最低下的那个选项（扩展程序），或者使用快捷键Ctrl+Shift+X，输入view in browser，安装即可，安装完成，右键index.html文件即可看到view in browser

8、添加sass支持：https://www.sass.hk/skill/sass154.html
打开VSCode编辑器，然后按ctrl+P，键入ext install live-sass，安装Live Sass Compiler即可

参数

"liveSassCompile.settings.formats":[
	// 扩展
	{
		"format": "compact",//可定制的出口CSS样式（expanded，compact，compressed，nested）
		"extensionName": ".min.css",//编译后缀名
		"savePath": null//编译保存的路径
	}
],
"liveSassCompile.settings.excludeList": [
	"**/node_modules/**",
	".vscode/**"
],

8.1、按F1或ctrl+shift+P键入Live Sass: Watch Sass以开始实时编译，或者按键入Live Sass: Stop Watching Sass以停止实时编译。

8.2、按F1或ctrl+shift+P键入一次Live Sass: Compile Sass - Without Watch Mode 以编译Sass或Scss。

9、添加autoprefixer扩展：点击最下面的那个图标（扩展程序），搜索autoprefixer，直接安装是3.0.1版本的autoprefixer，安装好了之后点击插件的设置，点击安装另一个版本...，安装2.2.0版本的autoprefixer

9.1、设置settings.json：打开编辑器的设置（左上角文件-->首选项-->设置），搜索autoprefixer，然后在Autoprefixer: Browsers下面点击在settings.json中编辑

9.2、settings.json添加如下代码

"autoprefixer.browsers": [
        "ie >= 6",  //ie6以上
        "firefox >= 8",
        "chrome >= 24",
        "Opera>=10"
],

9.3、新建一个css文件（文件名可以自己取，我这边示例取名为autoprefixer.css），写一下css3中的display: flex;（未写兼容的），然后按下组合键Ctrl+Shift+P，输入auto，点击Autoprefix CSS编译这个css文件，就可以给css文件中未兼容的代码加上兼容前缀了

9.4、在sass中使用autoprefixer：打开设置（没什么好说的），搜索sass，点击扩展中的Live Sass Compile，安装了autoprefixer之后会在Live Sass Compile中多出来一个选项Live Sass Compile › Settings: Autoprefix，点击在在settings.json中编辑

在settings.json中添加如下代码，执行Live Sass: Watch Sass将sass文件编译为css文件的时候生成的css文件就可以带上兼容属性了

"liveSassCompile.settings.autoprefix": [
        "ie >= 6",  //ie6以上
        "firefox >= 8",
        "chrome >= 24",
        "Opera>=10"
],
